# 横版卷轴游戏教程


本教程已集成到Defold编辑器中，可以轻松访问：

1. 启动Defold。
2. 在左侧选择*新建项目*。
3. 选择*来自教程*选项卡。
4. 选择"横版卷轴游戏教程"。
5. 在本地驱动器上选择项目的位置，然后点击*创建新项目*。

![新建项目](images/new-side-scroller.png)

编辑器会自动从项目根目录打开"README"文件，其中包含完整的教程文本。
