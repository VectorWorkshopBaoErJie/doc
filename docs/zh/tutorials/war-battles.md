# 战争战斗教程

在本教程中，你将学习如何创建一个具有移动和射击机制的小型可玩游戏。

本教程已集成到Defold编辑器中，易于访问：

1. 启动Defold。
2. 在左侧选择*新建项目*。
3. 选择*来自教程*选项卡。
4. 选择"战争战斗教程"
5. 在本地驱动器上为项目选择一个位置，然后点击*创建新项目*。

![new project](images/new-war-battles.png)

编辑器会自动从项目根目录打开"README"文件，其中包含完整的教程文本。

[你也可以在Github上阅读完整的教程文本](https://github.com/defold/tutorial-war-battles)
